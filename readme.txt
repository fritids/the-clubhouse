The clubhouse is a system that allows disc golf clubs to capture player information, manage events, and more.

== Description ==

With the clubhouse, disc golf clubs can manage player information and events. A registration management system provides means to track players attending events, manage scores, and output leaderboard information.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
